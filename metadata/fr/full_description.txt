<p>My Price Log vous aide à enregistrer et comparer les prix de vos magasins locaux. L'application marche entièrement hors-ligne:</p><ul><li>Vous entrez une liste de magasins.</li><li>Vous entrez la liste des produits vendus dans ces magasins.</li><li>Vous entrez, mettez à jour, et confirmez les prix de ces produits quand vous faites les courses.</li><li>L'application vous montre les prix à l'unité des produits vendus dans chaque magasin, vous permettant de décider où acheter quel produit.</li></ul><br><br><p>Ici il n'y a rien de particulièrement élégant ou intelligent. C'est simplement une base de données spécialisée dotée d'une interface utilisateur optimisée pour une utilisation facile quand on est debout dans une allée de supermarché, avec un caddie et/ou un sac dans les mains.</p><br><br><p>L'application indique la date d'entrée du prix, de sorte à ce que vous ne soyez pas induit en erreur par des prix obsolètes. Les vieux prix sont indiqués par des icônes, et une ajustation (simplifiée) causée par l'inflation est appliquée pour tenter de les rendre à peu près utiles, bienque pour de meilleurs résultats vous devriez mettre-à-jour les prix vous-mêmes. You can view a price’s history if you want to see how it has changed over time.</p><br><br><p>The app will make a recommendation as to whether a price is good, OK or bad if it has enough information. A list of unit prices is also shown so you can decide for yourself.</p><br><br><p>Customer loyalty scheme discounts can be specified for each store and these will be used to adjust the unit price when making a comparison between stores.</p><br><br><p>Unit conversions are handled automatically, both within and between measurement systems. Metric, imperial and US customary units are all supported.</p>
